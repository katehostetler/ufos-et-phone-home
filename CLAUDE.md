# UFOs ET Phone Home — Project Instructions

## Project Overview
This is the `ufos-et-phone-home` project. All contributors (human and AI) should follow these rules in addition to the global CLAUDE.md at `~/.claude/CLAUDE.md`.

## Project Rules

### Documentation Requirements
- **CHANGELOG.md** must be updated with every change (date + summary of what changed)
- **README.md** must be updated whenever the project structure, purpose, or setup instructions change
- Both updates should be included in the same commit as the code change — not as an afterthought

### File Organization
- Keep the project root clean — use subfolders for content categories as they emerge
- Document any new folder structure in the README

### Content Integrity
- Never fabricate content, metrics, URLs, or data
- If referencing external data, note the source

### Git Workflow (project-specific)
- Branch naming: `feat/`, `fix/`, `content/`, `docs/` prefixes
- Commit messages: conventional commits (feat:, fix:, docs:, content:, style:)
- Commit after every working state
- Never commit directly to main (except the initial project setup)
